.. _About Python:

Über Python
============

Ein paar Fakten
---------------

* Erste Version von Guido van Rossum am 20.02.1991 veröffentlicht
* Namensherkunft: Monty **Python**’s Flying Circus
* Prinzipien der Sprache (veröffentlicht 1999):

  * an easy and intuitive language just as powerful as those of the major competitors
  * open source, so anyone can contribute to its development
  * code that is as understandable as plain English
  * suitable for everyday tasks, allowing for short development times

* Aktuell 3\. Platz der TOP Ten TIOBE Sprachen (We are coming for you, Java!)

  * 1\. Platz Java
  * 2\. Platz C


Einsatzgebiete
--------------

* Automation
* (Remote) Konfigurieren von Servern

  * `Salt Stack <https://www.saltstack.com/>`_
  * `Ansible <https://www.ansible.com/>`_

* Künstliche Intelligenz

  * `Tensorflow <https://www.tensorflow.org/>`_
  * `PyTorch <https://pytorch.org/>`_

* Data Analysis

  * `NumPy <https://numpy.org/>`_
  * `Pandas <https://pandas.pydata.org/>`_
  * `SciPy <https://www.scipy.org/>`_

**Hinweis** Python ist keine schnelle Sprache (relativ!) und daher für Anwendungen, die maximale
Performance erfordern (zum Bsp. Videospiele, "Number Crunching") ungeeignet. Wer trotzdem
nicht auf Python verzichten möchte, für den gibt es Kompromisse:

* `Cython <https://cython.org/>`_
* `Stackless <https://stackless.readthedocs.io/en/3.6-slp/stackless-python.html>`_
* `Jython <https://www.jython.org/>`_
* `PyPy <https://www.pypy.org/>`_


Die Sprache
-----------

* Einfach zu lernen
* Durchdacht und konsistent (es funktioniert so, wie man es erwartet)
* "Batteries included!" (große stdlib)
* `PEP8 <https://www.python.org/dev/peps/pep-0008/>`_ und `PEP20: Zen of Python <https://www.python.org/dev/peps/pep-0020/>`_
* Sehr mächtig
* Und einfach magisch!
