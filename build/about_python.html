Über Python¶

Ein paar Fakten¶

  • Erste Version von Guido van Rossum am 20.02.1991 veröffentlicht

  • Namensherkunft: Monty Python’s Flying Circus

  • Prinzipien der Sprache (veröffentlicht 1999):

    • an easy and intuitive language just as powerful as those of the major competitors

    • open source, so anyone can contribute to its development

    • code that is as understandable as plain English

    • suitable for everyday tasks, allowing for short development times

  • Aktuell 3. Platz der TOP Ten TIOBE Sprachen (We are coming for you, Java!)

    • 1. Platz Java

    • 2. Platz C

Einsatzgebiete¶

  • Automation

  • (Remote) Konfigurieren von Servern

    • Salt Stack

    • Ansible

  • Künstliche Intelligenz

    • Tensorflow

    • PyTorch

  • Data Analysis

    • NumPy

    • Pandas

    • SciPy

Hinweis Python ist keine schnelle Sprache (relativ!) und daher für Anwendungen, die maximale Performance erfordern (zum Bsp. Videospiele, „Number Crunching“) ungeeignet. Wer trotzdem nicht auf Python verzichten möchte, für den gibt es Kompromisse:

  • Cython

  • Stackless

  • Jython

  • PyPy

Die Sprache¶

  • Einfach zu lernen

  • Durchdacht und konsistent (es funktioniert so, wie man es erwartet)

  • „Batteries included!“ (große stdlib)

  • PEP8 und PEP20: Zen of Python

  • Sehr mächtig

  • Und einfach magisch!
